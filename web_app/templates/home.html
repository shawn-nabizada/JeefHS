Hybrid Edge/Cloud Architecture

JeefHS isn't just a tracker; it's a distributed system designed for reliability.

The system operates on a Local-First principle. The Raspberry Pi (Edge) manages immediate security threats and device control locally to ensure zero latency. Data is synchronized with the Cloud (Neon Postgres) only when connectivity allows, ensuring no data loss during internet outages.
